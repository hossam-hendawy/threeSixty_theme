== Description ==

You can reload Gravity Forms on AJAX submission. Useful in situations where you can allow multiple form submissions without refreshing the page.
Also, you can add a link to refresh the form at the bottom of the thank-you message by using the '{reset_form}' parameter in the Gravity Form's confirmation settings, clicking on which will reset the form.

Features of Form Refresher for Gravity Forms:
   * Reloading the form without refreshing the page.
   * Can set auto to reload time.
  

If you come across any bugs or have any suggestions, please email or contact us at vivek.jha@galaxyweblinks.in


== Installation ==

Installation steps

1. Upload the folder to `/wp-content/plugins/` directory
2. Activate the plugin via the 'Plugins' menu in WordPress
3. To enable the plugin options go in the admin Form Refresher for Gravity Forms menu


==Frequently Asked Questions==

=How to use Form Refresher for Gravity Forms Plugin?

In Gravity Forms settings, you can set the time by activating Automatically Reset Form, and you can save the settings by adding the parameter '{reset_form}' from the confirmation setting to add the reset link.
Example: wp-admin > Forms > Settings
